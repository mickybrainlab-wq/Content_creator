# PART 4 — THE COMPLETE CONTENT FRAMEWORK

## How to Create Any Survival Content — Step by Step

```
┌─────────────────────────────────────────────────────────────────┐
│              THE CONTENT FRAMEWORK                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  STEP 1 — IDENTIFY THE FEAR                                   │
│  └── What is the customer AFRAID of?                        │
│  └── Example: "Losing everything in an emergency"          │
│                                                                  │
│  STEP 2 — CREATE THE CONTRAST                                │
│  └── Show life BEFORE (unprotected, unprepared)            │
│  └── Show life AFTER (safe, secure, protected)            │
│                                                                  │
│  STEP 3 — POSITION THE PRODUCT                              │
│  └── The product is the SHIELD against the threat         │
│  └── "With this, you become protected"                   │
│                                                                  │
│  STEP 4 — WRITE THE WORDS                                  │
│  └── Describe the THREAT, not the product               │
│  └── Make it feel URGENT, REAL, IMMEDIATE              │
│                                                                  │
│  STEP 5 — ADD THE SURVIVAL FLAVOR                         │
│  └── Urgency: "What if it happens TODAY?"             │
│  └── Direct words: Protect, Secure, Shield, Guard   │
│  └── The moment: The crisis, the disaster          │
│                                                        │
└─────────────────────────────────────────────────────────────────┘
```
